# rs-pdf-example

## Version

`0.1.0`

## Usage

```
Usage: rs-pdf [OPTIONS] --input <INPUT_PATH> --output <OUTPUT_PATH>

Options:
  -i, --input <INPUT_PATH>    
  -o, --output <OUTPUT_PATH>  
  -d, --debug                      
  -a, --allow-override             
  -h, --help                       Print help
  -V, --version                    Print version
```
