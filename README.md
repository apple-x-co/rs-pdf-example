# rs-pdf-example

## Usage

```
Usage: rs-pdf [OPTIONS] --input-path <INPUT_PATH> --output-path <OUTPUT_PATH>

Options:
  -i, --input-path <INPUT_PATH>    
  -o, --output-path <OUTPUT_PATH>  
  -d, --debug                      
  -a, --allow-override             
  -h, --help                       Print help
  -V, --version                    Print version
```
